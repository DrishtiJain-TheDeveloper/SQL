# SQL Student Management System

This project demonstrates the use of SQL for managing student records in a database system.

## Features
- Create student database
- Store student information
- Manage course details
- Track student marks and attendance
- Perform CRUD operations
- Execute JOIN and aggregate queries

## Technologies Used
- SQL
- MySQL

## SQL Concepts Used
- CREATE TABLE
- INSERT
- SELECT
- UPDATE
- DELETE
- JOINS
- GROUP BY
- ORDER BY
- Aggregate Functions

## Project Objective
The objective of this project is to demonstrate practical knowledge of SQL database management and query execution through a student management system.

## Sample Operations
- Add new students
- Update student records
- Fetch topper details
- Display students course-wise
- Calculate average marks
